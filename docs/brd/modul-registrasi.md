### Data Identitas Pasien

|Nama Field|Sifat|Alasan / Catatan|
|-|-|-|
|No. Rekam Medis (RM)|Mandatory|Generated by system secara otomatis.|
|Nama Lengkap|Mandatory|Sesuai KTP/KK untuk menghindari salah input obat.|
|NIK / No. Pasien|Mandatory|Kunci utama untuk integrasi (BPJS/SatuSehat).|
|Tanggal Lahir|Mandatory|Vital untuk dosis obat dan verifikasi identitas.|
|Jenis Kelamin|Mandatory|Penting untuk diagnosis medis dan pendaftaran poli.|
|No. Telepon / WhatsApp|Mandatory|Untuk konfirmasi kedatangan atau pengingat kontrol.|
|Alamat Domisili|Mandatory|Minimal Kota/Kabupaten & Kecamatan untuk pelaporan dinas.|
|Pekerjaan|Opsional|Tambahan informasi sosiodemografi.|
|Email|Opsional|Untuk pengiriman invoice digital atau hasil lab.|

### Data Kunjungan & Klinis Awal

|Nama Field|Sifat|Alasan / Catatan|
|-|-|-|
|Jenis Penjamin|Mandatory|Umum, BPJS, atau Asuransi Swasta (menentukan alur berkas).|
|No. Kartu Asuransi|Kondisional|Wajib diisi jika penjaminnya bukan "Umum".|
|Poli Tujuan|Mandatory|Mengarahkan antrean ke dokter yang tepat.|
|Dokter Spesialis|Mandatory|Menentukan antrean dokter spesifik.|
|Keluhan Utama|Mandatory|Ringkasan singkat (misal: "Demam 3 hari", "Kontrol hamil").|
|Alergi Obat|Mandatory|Sangat Vital! Harus ada checkbox "Tidak Ada" atau kolom teks jika ada.|
|Nama Kontak Darurat|Opsional|Sangat berguna untuk kasus emergency.|
|No. Tlp Kontak Darurat|Opsional|Setali tiga uang dengan nama kontak darurat.|

### Format Antrean: [Kode Poli/Layanan] - [Nomor Urut]
- A (Umum/Gigi): A-001, A-002 (Untuk Poli Umum atau Gigi)
- B (Spesialis Anak): B-001, B-002 (Untuk Poli Anak)
- C (KIA/Kebidanan): C-001, C-002
- X (Khusus/Prioritas): X-001 (Untuk lansia, disabilitas, atau ibu hamil besar)
